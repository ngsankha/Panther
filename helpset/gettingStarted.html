Panther

This is just a guide to get you started with Panther in no time.

Encode your Music Files:

You will see a window as shown below:

Just select the file you want to encode as the input file by clicking Browse beside it. Select the file you want to save it as after the encoding process as the output file by clicking Browse. Then select the the format to which you want to encode from the dropdown list and just hit Encode. And you’re done!

Reduce the size of your MP3 files:

You will see the window as shown below:

Just add the files to the list you want to encode by clicking Add. You can know the current bitrate of a file by selecting that file from the list and clicking on Find Bitrate. After that just click Browse and select the output folder, ie. the folder in which you want to keep the files of reduced size! Then just click Encode.

 Encode your video files:

Select the presets from the list of available presets, ie. the format to which you want to convert your video files and then click Next. Then select the input file and output folder in the next window. You can set the other options according to your wish but you will not need to, the preset fills the default parameters for the encoding process by itself! After that just click Encode.
