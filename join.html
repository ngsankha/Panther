Join the Panther Project.

 

Loved it till here? Want to contribute?

 

Want to help with the development of Panther? Here I'll list some of the ways in which you can do so.

 

Develop your own code

If you want to contribute to the codebase of Panther by write some code, then you can do the following steps:

  • Fork the Panther Project from the Github repository here.
  • Develop your additions to the code.
  • Send me a Pull Request on Github for merging your code with main repository.

Detailed help on these topics is available on the Github Help.

 

Writing Documentation

People interested in writing the help documents for Panther can also join. Panther is still undocumented. To do so just drop me a mail at sankha93@gmail.com!

 

For Githubbers

If you are on Github then you can follow me at https://github.com/sankha93.

If to keep yourself updated on the latest developments on the Panther Project you can follow it at https://github.com/sankha93/Panther.

 

None of the above? No worries.

If you still want to help then you can promote Panther. Tell your friends, share links on Facebook, Twitter! Even blog about Panther.
